# 🎮 坦克大战 (Tank Battle)

一个用纯 HTML5 Canvas 和 JavaScript 实现的经典坦克大战游戏。

## 🎯 游戏特色

- **经典玩法**：控制你的坦克，消灭所有敌人，保卫基地！
- **多个关卡**：3个精心设计的地图，难度递增
- **道具系统**：速度提升、快速射击、护盾、额外生命
- **粒子特效**：爆炸、射击、墙体破坏等视觉效果
- **音效系统**：基于 Web Audio API 的 8-bit 风格音效
- **移动端支持**：响应式设计，支持触屏操作

## 🎮 操作说明

| 按键 | 功能 |
|------|------|
| `↑↓←→` / `WASD` | 移动坦克 |
| `空格` / `J` | 射击 |
| `P` | 暂停/继续 |
| `R` | 重新开始 |

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏！

```bash
# 或者使用任意 HTTP 服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 🛠️ 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES6+)
- Web Audio API
- CSS3 动画

## 📝 游戏规则

1. 消灭所有敌方坦克即可过关
2. 保护你的基地（⭐标记），基地被摧毁则游戏结束
3. 你有 3 条生命，被敌人子弹击中会失去一条命
4. 收集道具获得临时增益效果
5. 每关敌人数量递增，难度逐渐加大
